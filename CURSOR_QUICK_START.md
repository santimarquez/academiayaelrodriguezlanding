# CURSOR Quick Start

## Example Prompts

- "Add a new section below the hero with the academy's value proposition"
- "Implement the testimonials section with card-dark-translucent styling"
- "Add JSON-LD schema for LocalBusiness"
- "Replace the hero portrait placeholder with an ImagePlaceholder component"
- "Improve meta tags for SEO"
- "Add a blog index page using Content Collections" (when blog is ready)

## Common Workflows

1. **New section:** Check `ai/context/PROJECT_STRUCTURE.md`, create component in `src/components/`, import in `index.astro`
2. **Design tokens:** Use academy-pink, academy-dark; fonts: Playfair Display, Montserrat
3. **Placeholder images:** Use `ImagePlaceholder.astro` with appropriate color and aria-label
4. **After changes:** Update `ai/context/RECENT_CHANGES.md`

## Reference

- `.cursorrules` – project rules
- `ai/context/` – structure, patterns, standards

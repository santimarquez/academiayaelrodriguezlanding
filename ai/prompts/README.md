# AI Prompts

Prompt templates for common tasks in this project.

## Purpose

Reusable prompts for:
- Adding new sections
- SEO improvements
- Component refactors
- Blog implementation (when ready)

## Usage

Copy and adapt prompts from this folder when asking Cursor to perform tasks.

## Adding prompts

1. Add a new .md file with a descriptive name
2. Include: context, task, expected output, optional constraints
3. Mention in this README

# CURSOR – Academia Yael Rodríguez Landing

## Context

Landing page for Yael Rodríguez Beauty & Business Academy. Static site built with Astro + Tailwind. Spanish-only. Blog-ready (Content Collections configured, blog not implemented yet).

## Site Structure

1. **Hero** – H1 "Yael Rodríguez", tagline, CTA, portrait
2. **Philosophy** – Centered copy, dark overlay (floral-bg)
3. **Differentiation** – Value prop, 2 images, method pillars
4. **Courses** – Nails Pro hero, 3 cards (Cejas, Dental, Mirada), "Más líneas formativas"
5. **Testimonials** – 3 cards, Google CTA
6. **CTA Decision** – "El momento de decidir es ahora"
7. **Academic Team** – Yael, Tania, Nerea
8. **Contact** – Próxima Convocatoria, form CTA, Tel/Email/Dirección
9. **Salon Footer** – Full-width image, accessibility note

## Tech

- **Astro** – static output, Content Collections for future blog
- **Tailwind** – academy-pink (#e298b4), academy-dark (#1a1a1a)
- **Fonts** – Playfair Display, Montserrat

## Run

```bash
npm install
npm run dev      # local dev
npm run build    # production build
```
